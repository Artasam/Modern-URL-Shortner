# Modern URL Shortener - Frontend (Vite)

This directory contains the Feature-Sliced UI and global logic representing the client face of the **Modern URL Shortener**.

🌍 **Live App Link**: [https://modern-url-shortner.vercel.app/](https://modern-url-shortner.vercel.app/)

## Key Technologies
- **React 19**
- **Vite 6**
- **Tailwind v4**
- **Three.js & React-Three-Fiber**
- **TanStack Query**

## Deploying to Vercel
This project uses Vercel for completely free, edge-optimized hosting:
1. Push your repository to GitHub.
2. In your Vercel Dashboard, import the repository.
3. Make sure to set the **Root Directory** setting to `frontend`.
4. Ensure the Framework Preset is detected as **Vite**.
5. Assign a `VITE_API_URL` environment variable pointing to the remote backend (e.g. PythonAnywhere server).

*For complete architectural information, refer to the [Root README](../README.md).*
